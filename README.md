# DApp_DBANK

A decentralized banking application built with modern web technologies.

## Contents

This repository contains the frontend assets for the DBank DApp:
- HTML and CSS files for the user interface
- Logo and branding assets
- Styling and layout components

## Documentation

- **[How to Push Large Folders to GitHub](PUSHING_LARGE_FOLDERS.md)** - Complete guide for handling large files and folders when pushing to GitHub from Git Bash

## Getting Started

The frontend files are located in the `Adding HTML and CSS to Create the Frontend for DBAN` directory and include:
- `index.txt` - HTML structure for the DBank interface
- `main.txt` - CSS styling for the application
- `dbank_logo.png` - DBank logo asset

## Contributing

When contributing large files or folders to this repository, please refer to the [Pushing Large Folders Guide](PUSHING_LARGE_FOLDERS.md) for best practices.

## Git LFS

This repository is configured to use Git Large File Storage (LFS) for handling large binary files. Ensure you have Git LFS installed before pushing large assets.
